# Security Policy

## Reporting a Vulnerability

Please report security issues privately by email:

- `yepuri.sasi07@gmail.com`

Include:

- Impact summary
- Steps to reproduce
- Affected files/components
- Suggested mitigation (if available)

## Security Practices in This Repository

- Do not commit `.env` or API keys.
- Runtime artifacts and local databases are git-ignored.
- Prefer local/Ollama mode for development when possible.
- Review logs before sharing to avoid exposing sensitive customer data.
